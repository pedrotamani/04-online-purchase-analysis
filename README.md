# 🛒 Online Purchase Exploratory Data Analysis

This project performs an **Exploratory Data Analysis (EDA)** on an online purchase dataset using Python.
The goal is to explore purchasing patterns, product categories, pricing behavior, and payment preferences through data visualization and statistical exploration.

This project is part of a **Data Analytics portfolio**, demonstrating data cleaning, exploration, and visualization techniques using Python.

---

# 📊 Project Objectives

* Explore patterns in online purchase behavior
* Analyze the distribution of product prices
* Identify the most common product categories
* Evaluate seller ratings and performance
* Analyze payment method usage
* Explore the geographical distribution of purchases

---

# 🗂️ Dataset

The dataset contains information about online purchases including product details, sellers, payment methods, and geographic coordinates.

### Dataset Columns

* Product
* Product Category
* Price
* Shipping Cost
* Purchase Date
* Seller
* Purchase Location
* Rating
* Payment Method
* Installments
* Lat
* Lon

Some observations from the analysis include:

* Certain product categories concentrate a higher number of purchases.
* Product prices show noticeable variability across categories.
